Eerste website

Toen Tim Berners-Lee samen met Robert Cailliau het wereldwijde web uitvond, maakte Tim Berners-Lee gelijk de eerste website ooit. Deze website is hier rechts te zien. De eerste website ooit is nog heel simpel. Bovenaan staat de titel, daaronder staan veel hyperlinks. Met deze hyperlinks kan je verschillende websites met elkaar verbinden. Als je op een van die links klikt wordt je doorgestuurd naar een andere pagina, deze pagina staat ook weer vol met andere links. In 1990 bouwde Berners-Lee de eerste website ooit. Hij lanceerde deze website op 6 augustus 1991. De website heet info.cern.ch, de website staat nog steeds online.
